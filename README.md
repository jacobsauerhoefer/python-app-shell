## Installation

```bash
#Install Virtual Environment:
virtualenv venv

#Enter Virtual Environment
source venv/bin/activate

#Install Dependencies:
pip install -r requirements.txt

#Start Server
python server.py
```

To cleanup *.pyc files:

`sh cleanup.bash`
